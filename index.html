Picking Products In AGV

Method : POST

Url :http://172.18.0.135:8012/AGVadmin/pickingInAGV

Body : JSON 

{
            "billNumber": "JIB620909-001",
            "billType": "JIB-Online-Order",
            "priority": "2 hr",
            "shipToName":"Mr.Somyot Chaowalit",
            "remark":"Order ONLINE babababa",
            "details": [
                {
                    "skuCode": "8905533028",
                    "skuName": "MICROPHONE STAND NUBWO M21 GOLD 1-Y",
                    "quantity": 2
                },
                {
                    "skuCode": "8905533041-1",
                    "skuName": "K/B FANTECH GAMING HUNTER PRO [K511] 2-Y",
                    "quantity": 2
                }
            ]
        }


Data Dictionary

Prefix Notes
billNumber ออร์เดอร์ของออนไลน์
billType ประเภทของรายการ เช่น JIB-Online-Order(บิลของออนไลน์) , Lazada-Online-Order
priority ความสำคัญของออร์เดอร์ เช่น 1.30 hr ,same day หรือ nextday เป็นต้น
shipToName ชื่อลูกค้า หรือ คนสั่งออร์เดอร์
remark หมายเหตุ หรือข้อความที่ต้องการบอกคนหยิบ
skuCode รหัสสินค้า
skuName ชื่อสินค้า
quantity จำนวนหยิบ
OUT
statusBill สถานะบิลที่ได้จาก AGV {true=>กำลังหยิบให้,false=>ไม่สามารถทำงานได้ให้ดูที่ msg}
inAgv รายการสินค้าที่มีใน AGV
notAgv รายการสินค้าที่ไม่มีใน AGV

Picking Cancel (ยกเลิกออร์เดอร์)

Url :http://172.18.0.135:8012/AGVadmin/pickingCancel

Body : JSON

              "billNumber":"JIB620909-001"
